Самый простой способ убрать картинки из боковой панели "Файлы" — установить специальный плагин.

### ✅ Explorer Hider (рекомендую)

Этот плагин позволяет скрывать любые файлы и папки из проводника прямо через контекстное меню [](https://github.com/Mara-Li/obsidian-explorer-hider)[](https://www.obsidianstats.com/plugins/explorer-hider)

**Как использовать:**

1. Установите плагин **Explorer Hider** из сообщества Obsidian
    
2. Нажмите правой кнопкой мыши на папку с картинками (например, `Attachments`)
    
3. Выберите **"Hide folder"**
    
4. Папка исчезнет из файлового дерева
    

**Плюсы:** работает просто, можно в любой момент вернуть видимость через команду `Explorer Hider: Toggle Visibility`.
